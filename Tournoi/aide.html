Comment utiliser le catalogue

Comment naviguer dans les tournois ?

Utilisez les filtres en haut de page pour trier les tournois par type (Swiss, Élimination, etc.). Cliquez sur une carte pour voir les détails du tournoi.

Comment voir la liste des participants ?

Cliquez sur le bouton "Voir Participants" sur chaque carte de tournoi pour afficher la liste complète des participants.

Comment accéder au site officiel du tournoi ?

Utilisez le bouton "Site du Tournoi" sur chaque carte pour ouvrir le site officiel dans un nouvel onglet.

Problèmes fréquents

L'iframe ne se charge pas

Si l'iframe ne se charge pas correctement :

  • Vérifiez votre connexion internet
  • Assurez-vous que JavaScript est activé
  • Essayez de rafraîchir la page
  • Certains sites peuvent bloquer l'intégration en iframe

Les participants ne s'affichent pas

Si la liste des participants est vide :

  • Le tournoi vient peut-être d'être créé
  • Les inscriptions peuvent être encore en cours
  • Contactez l'organisateur du tournoi
